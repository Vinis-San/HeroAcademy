Plataforma Gamificada de Inglês
Visão Geral
A Plataforma Gamificada de Inglês é um projeto educativo que visa melhorar a leitura e escrita em inglês de seus usuários. Utilizando uma abordagem envolvente, os alunos assumem identidades de personagens como cavaleiros ou magos, e conforme completam provas de nível, eles avançam em classes específicas e sobem de nível, tornando o aprendizado uma aventura épica.

Funcionalidades
Cadastro e Login: Usuários podem criar uma conta e fazer login para acessar suas informações e progresso.

Perfil do Personagem: Cada usuário pode escolher um personagem (Cavaleiro, Mago, etc.) e ver seu progresso em termos de classe e nível.

Provas de Nível: Provas interativas para avaliar a proficiência em leitura e escrita em inglês.

Sistema de Níveis: Conforme os usuários completam as provas, eles ganham XP e sobem de nível em suas respectivas classes de personagens.

Feedback e Recompensas: Feedback imediato sobre o desempenho e recompensas para motivar a continuidade do aprendizado.

Cada nível requer um certo número de pontos de experiência (XP) para ser alcançado. Por exemplo, cada nível pode exigir 100 XP para avançar.

Provas de Nível
Estrutura das Provas
As provas serão divididas em diferentes níveis de dificuldade, abrangendo vários aspectos da língua inglesa:

Leitura: Compreensão de textos, interpretação de significados, identificar ideias principais e detalhes.

Escrita: Construção de frases, uso de gramática correta, coerência e coesão textuais.

Exemplos de Provas
Leitura
Nível 1: Pequenos textos com perguntas de múltipla escolha sobre o conteúdo.

Nível 2: Textos mais longos com perguntas descritivas.

Nível 3: Artigos complexos com perguntas de interpretação crítica.

Escrita
Nível 1: Completar frases com as palavras corretas.

Nível 2: Escrever pequenos parágrafos sobre temas simples.

Nível 3: Redações sobre tópicos mais complexos com critérios específicos de avaliação.

Interações e Navegação

Instalação

Navegue até o diretório do projeto: cd plataforma-gamificada-ingles

Abra o arquivo index.html no seu navegador

Rodando no GitHub Pages
Adicione todos os arquivos ao repositório e faça o primeiro commit:

bash

git commit -m "First commit"
git push origin main

Na seção "GitHub Pages", selecione a branch main e a pasta raiz (/root) como fonte e salve.
