Sobre o Projeto Hero Academy

O Hero Academy é um projeto desenvolvido como parte do módulo final do curso de Front End do Senai Taguatinga. Criado em aproximadamente 6 dias, o projeto é funcional, mas tem potencial para ser escalado a níveis maiores. Diversos conhecimentos e habilidades adquiridos durante o curso foram aplicados e testados aqui.

Desenvolvimento

Durante o desenvolvimento do Hero Academy, eu (Vinícius) integrei várias tecnologias e ferramentas importantes, incluindo JSON para gerenciamento de dados, JavaScript para funcionalidade e interatividade, CSS para estilização e HTML para a estruturação da página. Este projeto demonstra o conhecimento prático adquirido no Módulo de Front-End e testa minha capacidade de criar soluções web completas e funcionais.

Afinal, o que é o Hero Academy?

O Hero Academy é mais do que um simples projeto; é uma plataforma gamificada de nivelamento de inglês, projetada para tornar o aprendizado da língua inglesa mais envolvente e eficiente. Atualmente, estamos com apenas 5 perguntas base, e de acordo com o nível de acertos, a classe do usuário muda.

Criador do Projeto

O Hero Academy foi desenvolvido por Vinicius Andrade, formado em Análise e Desenvolvimento de Sistemas (ADS) e atualmente participante do curso de Aprendizagem em Tecnologia da Informação no Senai Taguatinga.
