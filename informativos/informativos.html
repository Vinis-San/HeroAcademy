Sobre o Projeto Hero Academy

Hero Academy é um projeto desenvolvido como parte do módulo final do curso de Front End do Senai Taguatinga. Este projeto foi criado por uma turma de aprendizes do curso de Aprendizagem em Tecnologia da Informação, onde diversos conhecimentos e habilidades adquiridos durante o curso foram aplicados e testados.

Ao longo do desenvolvimento do Hero Academy, os alunos integraram várias tecnologias e ferramentas importantes, incluindo JSON para gerenciamento de dados, JavaScript para funcionalidade e interatividade, CSS para estilização e HTML para a estruturação da página. Este projeto é uma demonstração do conhecimento prático adquirido pelos alunos e sua capacidade de criar soluções web completas e funcionais.

O Hero Academy é mais do que um simples projeto; é uma plataforma gamificada de nivelamento de inglês, projetada para tornar o aprendizado da língua inglesa mais envolvente e eficiente. A plataforma oferece quizzes, desafios semanais, certificações digitais, vídeos educativos, e uma biblioteca de recursos, entre outras funcionalidades, proporcionando uma experiência completa de aprendizado.

Este projeto destaca o compromisso e a dedicação dos alunos em aplicar seus conhecimentos técnicos para resolver problemas reais e criar produtos inovadores. Agradecemos ao Senai Taguatinga por fornecer o suporte e os recursos necessários para tornar este projeto uma realidade.

O Hero Academy foi desenvolvido por Vinicius Andrade, formado em Análise e Desenvolvimento de Sistemas (ADS) e atualmente participando do curso de Aprendizagem em Tecnologia da Informação no Senai Taguatinga.
